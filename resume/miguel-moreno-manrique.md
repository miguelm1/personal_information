## Instrucciones para generar la versión PDF
1. Instalar [Pandoc](https://pandoc.org/) y una distribución LaTeX (por ejemplo, TeX Live).
2. Ejecutar el comando:
   ```bash
   pandoc miguel-moreno-manrique.md -o miguel-moreno-manrique.pdf --from markdown --template eisvogel --listings
   ```
   *(El template `eisvogel` es opcional; puede sustituirse por otro o eliminarse la opción `--template`.)*
3. Verificar que los enlaces e imágenes se renderizan correctamente antes de compartir el PDF.
