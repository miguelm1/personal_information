# Blog técnico y artículos

Este repositorio recoge enlaces y resúmenes de contenidos que publico sobre Inteligencia Artificial aplicada a telecomunicaciones y despliegues FTTH.

## Últimas publicaciones
- *(En edición)* "Automatización de reportes operativos con LLMs en Onivia" — Pipeline que integra orquestación en la nube y prompts controlados.
- *(Próximamente)* "Aplicando modelos de clasificación para priorizar incidencias en redes FTTH" — Notebook en construcción con comparativa de algoritmos.
- *(Próximamente)* "Uso de datos geoespaciales y heurísticas de coste para optimizar rutas de fibra" — Artículo técnico en formato Markdown.

## Estructura propuesta
- `notebooks/` — Versiones ejecutables en Jupyter/Colab de los experimentos mencionados (incluyendo pipelines de IA y automatización).
- `articles/` — Artículos en Markdown listos para publicar en GitHub Pages o plataformas externas.
- `assets/` — Imágenes, diagramas y visualizaciones que acompañan cada publicación.

## Cómo contribuir
1. Documentar cada publicación con fecha y contexto.
2. Incluir referencias a datasets o repositorios relevantes.
3. Mantener notebooks ligeros y con instrucciones de ejecución.

> **Nota:** cuando publiques un nuevo artículo, añade el enlace y un breve resumen al listado anterior para que sea fácil de encontrar desde el README principal.
